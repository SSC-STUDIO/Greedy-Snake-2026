# Greedy Snake Game 贪吃蛇游戏

A modern implementation of the classic Snake game with enhanced features and beautiful graphics.
经典贪吃蛇游戏的现代实现，具有增强功能和精美图形。

## Features 特性

- 🎮 Dual control system (Keyboard and Mouse) | 双重控制系统（键盘和鼠标）
- 🎨 Beautiful graphics and animations | 精美的图形和动画
- 🎵 Background music and sound effects | 背景音乐和音效
- ⚙️ Customizable settings (Volume, Difficulty, Sound) | 可自定义设置（音量、难度、声音）
- 🤖 AI-controlled snakes for added challenge | AI控制的蛇增加游戏挑战性
- 🌟 Dynamic food spawning system | 动态食物生成系统
- ⚡ Smooth snake movement and camera following | 流畅的蛇移动和相机跟随
- 🎯 Multiple difficulty levels | 多个难度等级
- 🎨 Modern UI with buttons and menus | 现代化的按钮和菜单界面

## Requirements 系统要求

- Windows OS | Windows操作系统
- C++ compiler (Visual Studio recommended) | C++编译器（推荐使用Visual Studio）
- EasyX Graphics Library | EasyX图形库

## Installation 安装说明

1. Clone the repository | 克隆仓库:
```bash
git clone https://github.com/yourusername/Greed-Snake-Reproduction.git
```

2. Open the project in Visual Studio | 在Visual Studio中打开项目

3. Make sure you have the following dependencies | 确保安装以下依赖:
   - EasyX Graphics Library installed | 已安装EasyX图形库

4. Build and run the project | 构建并运行项目

## Controls 控制方式

- **Arrow Keys**: Move snake | 方向键：控制蛇移动
- **Mouse**: Alternative control method | 鼠标：替代控制方式
- **Left Click**: Speed up snake | 左键点击：加速
- **Right Click**: Enable mouse control | 右键点击：启用鼠标控制
- **ESC**: Exit game | ESC：退出游戏
- **P**: Pause game | P：暂停游戏
- **S**: Resume game | S：继续游戏

## Game Features 游戏功能

### Main Menu 主菜单
- Start Game | 开始游戏
- Settings | 设置
- About | 关于
- Exit | 退出

### Settings 设置
- Volume control | 音量控制
- Difficulty selection (Easy, Normal, Hard) | 难度选择（简单、普通、困难）
- Sound toggle | 声音开关
- Save/Cancel options | 保存/取消选项

### Gameplay 游戏玩法
- Collect food to grow | 收集食物成长
- Avoid lava areas | 避开熔岩区域
- Compete with AI snakes | 与AI蛇竞争
- Manage speed and direction | 管理速度和方向
- Camera follows your snake | 相机跟随你的蛇
