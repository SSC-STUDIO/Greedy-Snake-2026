# Greed Snake 2025

## Project Overview
现代贪吃蛇游戏，C++ 实现，具有精美图形、流畅动画和智能AI对手。

## Tech Stack
- Language: C++ (Visual Studio)
- Build: .sln / .vcxproj
- Entry point: Main.cpp

## Development Rules
- Branch convention: work on `codex/ai-Greed-Snake-2025` branch
- Always update CHANGELOG.md on releases
- Build with Visual Studio or MSBuild
- Test gameplay changes before committing

## Code Style
- Follow existing C++ conventions
- Keep game logic in Core/ and Gameplay/ directories
- Resource files: .rc, .aps

## Key Paths
- `Core/` — core game engine
- `Gameplay/` — game logic
- `Main.cpp` — entry point
- `Greed-Snake-2025.sln` — Visual Studio solution
